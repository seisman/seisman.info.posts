判定某年为闰年的标准是：

-   能被 4 整除且不能被 100 整除的
-   能被 400 整除的

用 C 语言实现代码就是

``` C
if ((year % 100 != 0) && (year % 4 == 0) || (year % 400 == 0)) {
   // 输出该年是闰年;
} else {
   // 输出该年是平年;
}
```

一种更简单的写法是

``` C
#define ISLEAP(yr) ((!((yr) % 4) && (yr) % 100) || !((yr) % 400))
```
