Python 多版本共存之 pyenv

2013-10-04
编程

不再建议使用 pyenv，建议使用 Anaconda 。

经常遇到这样的情况：

  • 系统自带的 Python 是 2.6，自己需要 Python 2.7 中的某些特性；
  • 系统自带的 Python 是 2.x，自己需要 Python 3.x；

此时需要在系统中安装多个 Python，但又不能影响系统自带的 Python，即需要实现 Python 的多版本共存。pyenv 就是这样一个 Python 版本管理器。
